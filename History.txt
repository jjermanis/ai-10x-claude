Another game description was not followed. When levels change, the old exponent leaves and the 
new one remains. However, each were just moving a small amount. From the game description:
"When the game goes up 1 level, the exponent for 10 increases by 1, and a new random number is 
generated. If there is an exponent next to 10, it exits to the right. It will continue moving 
right until they move past the right end of the game screen. It travels fast enough to leave in 
1 second. When there is no exponent on the screen, the number for the new exponent comes from 
the beyond the right edge of the game screen and travels to the exponent spot next to the 10. 
Same as the other: the new exponent travels across in 1 second."
Claude came back with a high level description of how to fix. I asked it to be specific: provide
the code to use. I got that, applied it, it was easy and it worked. It's not exactly the same as

This looks like all the game description defects have been resolved. Next will be issues that
